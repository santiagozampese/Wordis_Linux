Word is a game similar to Termo, but it runs in the Linux Terminal.

To begin, give permission to the "install.sh" file and run it; it will create a game desktop.

Please, when starting the game, DO NOT CLOSE it before choosing the language, and DO NOT CHOOSE non-existent options. If necessary, delete the "linguas.json" file and start the desktop again.

If any files other than "linguas.json" are lost, please download the repository again.

If you have experience, you are permitted to modify the "words.json" file with RESPONSIBILITY.
